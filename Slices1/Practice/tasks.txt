Слайсы (Slices)
1. Базовые операции +
Напиши функцию SumSlice(slice []int) int, которая возвращает сумму всех элементов слайса.

2. Фильтрация +
Напиши функцию FilterEven(slice []int) []int, которая возвращает новый слайс только с чётными числами.

3. Реверс слайса +
Напиши функцию ReverseSlice(slice []int) []int, которая разворачивает слайс без создания нового (изменяя исходный).

4. Удаление дубликатов +
Напиши функцию RemoveDuplicates(slice []int) []int, которая возвращает новый слайс без повторяющихся элементов.

5. Слияние слайсов +
Напиши функцию MergeSortedSlices(a, b []int) []int, которая объединяет два отсортированных слайса в один отсортированный.

6. Скользящее окно +
Напиши функцию MaxSubarraySum(slice []int, k int) int, которая находит максимальную сумму подмассива длины k.

7. Динамическое программирование
Напиши функцию LongestIncreasingSubsequence(slice []int) int, которая возвращает длину самой длинной возрастающей подпоследовательности. (задача требует знания алгоритма бинарного поиска)
